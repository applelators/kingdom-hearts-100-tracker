# Kingdom Hearts Final Mix — 100% Tracker (Switch 2)

A spoiler-light, missable-aware **linear route tracker** for taking **Kingdom Hearts
Final Mix (KH1)** on Switch 2 to **Jiminy's Journal 100% in a single first-playthrough**,
on **Proud**.

Built as a sibling of `zelda-oot-100-tracker` / `xenoblade-100-tracker`: static SPA,
Cloudflare Worker + `[assets]`, all state behind `store.js`, zero backend, works offline
(a couch binge shouldn't depend on wifi).

## Source of truth

Mirrors **cR0Ck's GameFAQs "100% Story" walkthrough** (Kingdom Hearts HD I.5+II.5 ReMIX,
v4, 2023-09-07) chapter-for-chapter — an explicitly spoiler-free linear 100% route. The
tracker keeps the guide's **17 chapters** as phases and adds **checklist phases** for the
aggregate Journal systems (Trinities, Cups, 100 Acre Wood, Mushrooms, Clock Tower, Gummi,
Synthesis). **The in-game Jiminy's Journal % is the ultimate source of truth.**

## What "100%" means here

| Category | Count |
|---|---|
| 101 Dalmatians (puppies) | 99 (33 triad chests) |
| Trinity Marks | 46 (Blue 17 · Red 6 · Green 9 · Yellow 4 · White 10) |
| Ansem's Reports | 13 |
| Postcards | 9 (mailed in Traverse Town) |
| Olympus Cups | Phil / Pegasus / Hercules / Hades × (Normal · Solo · Time) |
| 100 Acre Wood | 5 pages + 6 minigame goals |
| White Mushroom masters | 7 magics (→ Dream Shield) |
| Neverland Clock Tower | 12 time-gated rewards |
| Gummi blueprints | 8 (Geppetto, by Heartless killed) |
| Synthesis | full recipe list → **Ultima Weapon** |
| Summons | Simba, Genie, Dumbo, Bambi, Tinker Bell, Mushu |
| **Superbosses** ★ | Phantom, Kurt Zisa, Unknown/Xemnas, Ice Titan, Sephiroth, Pot Scorpion |

Per-world generic chest grinds are a single "all N chests swept" checkbox each (so the
tracker isn't 280 potion rows); everything completion-relevant is itemized individually.

## Difficulty & the secret ending

Built for **Proud**, where the secret movie unlocks by **winning the Hades Cup**.
On **Standard** you instead need all 99 Dalmatians *and* a fully complete Journal
(Trinities, minigames, etc.); on **Beginner** the secret ending is impossible.

## How it's structured

- **Phases 1–17** = the guide's chapters (Destiny Islands → End of the World), with the
  three "100% Mastery" interludes and the superboss chapter (★ Endgame tier, visually set
  apart).
- **Checklist phases** = the Journal systems that have no single collection point.
- **Missables flagged in place (⚠)** — KH1 is forgiving (revisit any world before the
  finale), so flags are light: the **End of the World point-of-no-return door**, the
  ability-gated "come back after High Jump / Glide" pickups, the Ch.11 farm-before-it-gets-
  tedious window, the Vine-Jump Journal entry, and the recurring Clock Tower reminder.
- **Headline meters**: Dalmatians /99, Trinities /46, Ansem's Reports /13, plus overall %.
- **Strategy tips (💡)** on choice/skill moments — the permanent weapon + growth-speed
  picks, Riku's race, the Leon fight, and each Olympus Cup's gimmick.
- **"Up next" banner**, filters (hide-completed / only-missable / search), start countdown,
  export / import / reset.
- **Cross-device sync (☁)** — click *Sync*, enter a secret code, and use the same code on
  every device to share one save (stored in Cloudflare KV under `sha256(code)`; the raw
  code is never stored). localStorage stays the offline cache; the app pulls on load and on
  tab-focus, and pushes (debounced) on every change, with a last-write-wins `lastModified`
  stamp and a 409 guard so an older device can't clobber a newer save. Anyone with the code
  can read/write that save, so pick a non-obvious one.

## Data

`public/data/route.json` is generated by `build_route.py` (curated guide data + a
validator that asserts 33 puppy triads = 99, reports 1–13, 46 trinities, 12 clock doors,
9 postcards, and unique ids). Edit the script, not the JSON:

```bash
python3 build_route.py     # rewrites public/data/route.json, fails on any count mismatch
```

> Built pre-release on the HD Final Mix (1.5) baseline. Patch in any Switch 2 remaster
> changes (moved/added collectibles, postcard count) on launch night — it's all one file.

## Run it

```bash
# local preview (no install)
cd public && python3 -m http.server 8731   # → http://localhost:8731

# or with wrangler
wrangler dev
```

## Deploy (applelators Cloudflare account)

```bash
wrangler deploy
```

Pure static assets — no KV/D1 needed for v1. The Worker leaves a seam (`/api/*`) for a
future cross-device sync layer; only `worker.js` + `store.js` would change.

## Layout

```
worker.js              Worker entry (serves ./public via ASSETS)
wrangler.toml
build_route.py         data source → public/data/route.json (+ validation)
public/
  index.html
  styles.css           KH palette (deep blue / crown gold)
  store.js             localStorage persistence (checked / prefs)
  app.js               route rendering, meters, countdown, filters
  data/route.json      generated — the full 17-chapter route + checklist phases
```
